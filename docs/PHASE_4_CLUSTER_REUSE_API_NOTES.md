# Phase 4 Cluster Reuse API Notes

Last updated: 2026-06-05

## Scope

Historical note: this phase-4 quicklook endpoint was superseded by the plot-catalog update. Current formal Cluster web plots are regenerated by the backend into `/Volumes/Elements/satellite_data_web/outputs/generated_plots/cluster`; old B/E/POS quicklook PNGs under `/Volumes/Elements/data/idlpython_v2` are reference/debug artifacts only.

Phase 4 connected the backend to existing Cluster processed products under:

```text
/Volumes/Elements/data/idlpython_v2
```

The implementation is read-only:

- It scans existing `daily_full`, `daily_compact`, quicklook, manifest, and provenance files.
- It reads `daily_full` NPZ metadata/keys.
- It reads `daily_compact` parquet columns.
- At the time, it returned existing B/E/POS quicklook PNGs through an artifact route. Current API behavior returns `unsupported` for historical `existing_quicklook_*` plot requests and uses regenerated catalog plots instead.
- It does not read raw Cluster CDF content.
- It does not run `run_daily_production.py`.
- It does not modify frozen B-chain, E-chain, B/E ratio, IDL `.pro`, or `.sav` files.

## API Endpoints Used

```text
GET  /api/datasources/cluster/files?year=2005
GET  /api/datasources/cluster/metadata?file_id=20051203
GET  /api/datasources/cluster/variables?file_id=20051203
POST /api/datasources/cluster/plot
GET  /api/artifacts/{artifact_id}
```

Supported `plot_type` values:

- `existing_quicklook_b`
- `existing_quicklook_e`
- `existing_quicklook_pos`

Current replacement `plot_type` values are listed by `GET /api/datasources/cluster/plot-catalog`, including `cluster_magnetic_overview`, `cluster_electric_overview`, `cluster_spectrogram_overview`, `cluster_orbit_overview`, and `cluster_solar_wind_overview`.

Unsupported in phase 4:

- generated Cluster plots
- Cluster subset/export/stats
- production reruns
- formal L-bin/f-L/Figure 3/Cummings conclusions

## Cluster Variable Classification

The API labels `daily_full` variables conservatively:

- B and dB vectors: `magnetic_vector` or `magnetic`.
- E and dE vectors: `electric_vector` or `electric`.
- PSD matrices: `spectrogram`.
- L/MLT/MLAT fields: `context`.
- frequency axis: `frequency_axis`.
- B/E ratio fields: `be_ratio`.

These labels describe web display categories, not final science conclusions.
